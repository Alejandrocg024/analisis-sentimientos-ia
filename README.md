## About The Project


El proyecto tiene como objetivo desarrollar un modelo de aprendizaje automático para analizar el sentimiento en textos y predecir el tono emocional de tweets. 

Se utilizarán herramientas de procesamiento de lenguaje natural para generar un conjunto de datos de entrenamiento, depurar el texto recopilado y proponer un modelo que clasifique sentimientos en categorías específicas. La metodología incluye la recopilación de datos, limpieza del texto, etiquetado, validación manual, entrenamiento del modelo y predicción de sentimientos en tweets, con un enfoque final en analizar los estados de ánimo de dos personajes públicos relevantes en redes sociales.

Este proyecto ha sido realizado por Alejandro Campano Galán y Pablo Mera Gómez
